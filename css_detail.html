CSS Detail

CSS, yaani Cascading Style Sheets, ek aasan design language hai jo web pages ko presentable banane ka kaam karti hai. CSS se aap web pages par styles laga sakte hain. CSS ki madad se aap ye kaam HTML se alag kar sakte hain jo har web page ko banata hai. CSS ye batati hai ki webpage kaisa dikhega: ye color, fonts, spacing, aur bohot kuch prescribe karti hai.


CSS developers aur designers ko ye define karne deti hai ki kaise elements browser mein position ho. Jahan HTML tags ka istemal karta hai, wahan CSS rulesets ka istemal karti hai. CSS seekhna aur samajhna aasan hai, lekin ye HTML document ke presentation par powerful control provide karti hai.:


  1. * CSS time bachati hai: Aap ek baar CSS likh sakte hain aur wahi sheet ko multiple HTML pages mein reuse kar sakte hain.


  2. * Easy Maintenance: Global change karne ke liye bas style change kariye, aur saare elements saare webpages mein automatically update ho jayenge


  3. * Easy Maintenance: Global change karne ke liye bas style change kariye, aur saare elements saare webpages mein automatically update ho jayenge


  4. * Superior styles to HTML: CSS ke paas HTML se zyada wide array of attributes hote hain, isliye aap apne HTML page ko HTML attributes ke comparison mein behtar look de sakte hain


  5. * Offline Browsing: CSS web applications ko locally offline cache ki madad se store kar sakti hai. Iska istemal karke hum offline websites dekh sakte hain
